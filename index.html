🧭 Nizhny Novgorod Cultural Compass Telegram Bot

A smart bilingual Telegram bot that helps tourists and locals explore the historical landmarks of Nizhny Novgorod. By simply sharing their geolocation, users instantly discover the nearest point of interest with a description, distance, photo, and a direct link to Google Maps.

🌐 Bilingual English / Russian interface
📍 Geolocation Finds nearest landmark
📊 Analytics Live dashboard with stats
⚡ 24/7 Availability Hosted on Cloudflare
JavaScript Cloudflare Workers Telegram API Cloudflare KV Chart.js Haversine Formula
🤖 Try the Bot 📊 Live Dashboard 🐙 Source Code

📊 Analytics Dashboard

The bot collects real-time statistics: total users, /start commands, geolocation requests, and a leaderboard of the most popular landmarks. Data is stored in Cloudflare KV and visualized with Chart.js.
